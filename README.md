# for_shorthack
# AI-скрининг резюме: от 5000 откликов → к топ-20

## О проекте

Платформа для автоматического первичного отбора кандидатов с использованием ИИ.  
После отклика кандидат заполняет анкету, решает задачу (аналог LeetCode, но под его опыт) и проходит **VCV-интервью** с видеоаналитикой.  

**Результат:** скоринг + отсев 50% кандидатов без участия рекрутера.

---

## Ключевые функции

-  Генерация уникальной задачи под каждого кандидата (на основе анкеты)
-  Защита от читерства (блокировка вкладок, запись видео)
-  Анализ VCV через Whisper + GPT (речь → текст → оценка)
-  Ранжирование по трём метрикам: тест + VCV  + анкета 

## Реализованые алгоритмы:
```

1. TWO SUM (LeetCode)
   • Поиск двух чисел в массиве, дающих целевую сумму
   • Решение через HashMap за O(n)
   • Проверка решения кандидата через new Function()

2. ML-АНАЛИЗ ТЕКСТА
   • DistilBERT — классификация тональности
   • Поиск ключевых слов по 5 темам
   • Формула: (качество_речи × 0.6) + (темы × 1.5 × 0.4)

3. РАСШИФРОВКА РЕЧИ
   • Web Speech API — перевод видео в текст
   • Запасной генератор текста

4. РАНЖИРОВАНИЕ
   • Подсчёт итогового балла
   • Порог: ≥ 8 = прошёл, < 8 = перезапись
   • Сохранение истории попыток

5. ХРАНЕНИЕ ДАННЫХ
   • Сохранение результатов каждого этапа

6. ЕСТЬ МОБИЛЬНАЯ ВЕРСИЯ
```
